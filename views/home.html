AngularJS is working!

What is this?

This page was created to showcase AngularJS' capabilities by both exhibition (this page was made in AngularJS) and explanation on the process.

What this is not?

This is not a tutorial or guide on AngularJS (even if it may be a good introduction), neither is it made for publicity on the framework. Indeed, this is the second page on the "Works" series, the first one being all about Webpack.

So, what is going on in this page?

I will elaborate better in the process explanation page, but to briefly summarize, this is a Single Page Application (SPA) which takes a global data source (a ficticious user profile) which is used in three sections: that little widget on the header (saying "Hello" to you), the Do Something Cool! section, which also uses an exclusive data source, and, of course, the Profile Edit page, in which you add or change the user data seamlessly with your page. Your head would hurt thinking on how you'd do that without AngularJS or a similar tool, or, if you're an experienced developer, your head would instead hurt thinking about the amount of work.
